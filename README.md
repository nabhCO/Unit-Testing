# Unit-Testing

A project where I utilized C++ templates to implement basic data structures, then tested their functionality using the Catch2 framework. This repository features work that I completed for a class- in order to keep things fair for future students, have removed all details identifying the class and the specific project. This is also why this repository does not have a large commit history.

# Features

Skills demonstrated through this project:

- Generalized class implementation using templates, const, operator overloading, friend functions
- Design and implementation of unit tests
- Basic runtime analysis for basic data structures

# How to Run

The test file is run as part of the CI/CD pipeline for the project. To run tests in the terminal, enter:

1. make
2. ./test
